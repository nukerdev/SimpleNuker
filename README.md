# 🚀 SimpleNuker

Destroy discord servers (slow but it works)

---

## 📋 Requirements

- Python **3.12+**
- pip
- Git (optional, for cloning)

Install dependencies:

```bash
pip install -r requirements.txt
```

---

# 📱 Setup (Smartphone – Termux)

## 1️⃣ Installation

```bash
pkg install git
apt install python -y
git clone https://github.com/nukerdev/SimpleNuker.git
cd SimpleNuker
pip install -r requirements.txt
```

## 2️⃣ Configuration

Edit the `.json` file and insert your data.

## 3️⃣ Start

Normal:
```bash
python main.py
```

With screen session:
```bash
screen -S Simple-Nuker python main.py
```

---

# 💻 Setup (Computer)

## 1️⃣ Download

- Download the repository
- Extract the ZIP file
- Open the project folder

## 2️⃣ Install Requirements

### Windows
Double-click:
```
download_requirements.bat
```

### Linux
```bash
chmod +x download_requirements.sh
./download_requirements.sh
```

Or manually:
```bash
pip install -r requirements.txt
```

---

## 3️⃣ Configuration

Edit the `.json` file and insert your data.

---

## ▶️ Start

### Windows
Double-click:
```
start.bat
```

### Linux
```bash
chmod +x start.sh
./start.sh
```
